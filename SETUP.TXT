How to setup development environment

1) Install JDK
1) Instal maven
2) Install eclipse for Java developers with subclipse
3) Run 'mvn eclipse:eclipse' in this directory
4) Run eclipse
5) In eclipse settings add new JAVA classpath variable M2_REPO=~/.m2/repository
6) Import eclipse projects from this directory to eclipse
7) Donwload LWJGL v2.4.2 binaries from http://sourceforge.net/projects/java-game-lib/files/
8) Copy extracted binaries to /usr/lib/marten
9) When setuping any marten run configuration in eclipse add new environment variable 'LD_LIBRARY_PATH=/usr/lib/marten'

Happy hacking!
